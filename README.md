# Customer_behavior_analysis
Data analytic project showcasing customer behavior analysis using Python,Sql,Power Bi

🚀 Project Overview

This project presents a comprehensive end-to-end analysis of customer shopping behavior using transactional data. The objective is to uncover meaningful insights into **customer purchasing patterns, segmentation, and product performance** to support data-driven business decisions.

By combining **Python for data preprocessing**, **SQL for analytical querying**, and **Power BI for visualization**, this project simulates a real-world business intelligence workflow.

---

## 📊 Dataset Summary

* 📁 ~3,900 transaction records
* 🧾 18 features including demographics, purchase behavior, and product details
* ⚠️ Missing values handled using category-based median imputation

---

## 🛠️ Data Processing & Feature Engineering

* Cleaned and preprocessed data using **Pandas**
* Standardized column names for consistency
* Created new features:

  * `age_group` for segmentation
  * `purchase_frequency_days` for behavioral analysis
* Removed redundant columns to improve data quality
* Loaded cleaned data into **PostgreSQL** for analysis

---

## 📈 Key Business Analysis (SQL)

* Revenue analysis by gender and age group
* Customer segmentation (New, Returning, Loyal)
* Identification of high-spending discount users
* Top-performing products by rating and category
* Subscription vs non-subscription revenue comparison
* Shipping type impact on purchase behavior
* Discount dependency analysis

---

## 📊 Dashboard (Power BI)

An interactive dashboard was built to visualize:

* Revenue trends and KPIs
* Customer segmentation insights
* Product performance metrics
* Purchase behavior patterns

---

## 💡 Key Insights

* Subscribers generate higher average revenue
* A small segment of loyal customers contributes significantly to total sales
* Certain products rely heavily on discounts for sales
* Younger and middle-aged groups contribute the most revenue
